# .trae 协作规范治理入口

`.trae/` 是 Trae 的原生协作规范目录。本文件只说明 Trae 侧如何组织和维护规则。

## 读取顺序

1. 先读 `./.trae/runtime.md`。
2. 按任务类型读取 `./.trae/rules/*`。
3. 需要角色边界时读取 `./.trae/agents/*`。
4. 需要可复用流程时读取 `./.trae/skills/*/SKILL.md`。
5. 生成文档时读取 `./.trae/templates/*`。

## 维护方式

- `.trae/` 规则、角色、技能和模板单独维护，不从 `.ai/` 生成。
- 核心红线与 `.ai/` 保持一致，但具体文件内容按 Trae 使用体验单独优化。
- 修改 `.trae/` 后直接提交对应文件，不需要运行同步脚本。
